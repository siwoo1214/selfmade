# 개발 순서 계획

> 작성일: 2026-06-29
> 프로젝트: selfmade (식품 이커머스)

---

## 핵심 원칙

의존성 방향으로 순서를 잡는다.
나중에 만들 기능이 앞에서 만든 기능에 의존하도록 순서를 설계해야 개발 중 막히지 않는다.

---

## Phase 0 — 프로젝트 세팅

- [ ] 백엔드 패키지 구조 재정리 (도메인별로)
- [ ] React 프로젝트 초기 세팅 (`selfmade_front`)
- [ ] CORS 설정
- [ ] 공통 응답 포맷 정비 (`ApiResponse`)

---

## Phase 1 — 인증 / 회원

> 기존 `selfmade` 프로젝트에 구현된 코드를 활용한다.

- [x] 회원가입 / 로그인 / 로그아웃
- [x] JWT + Refresh Token Rotation (Redis)
- [ ] CUSTOMER / SELLER / ADMIN 역할 분리 확인
- [ ] SELLER 가입 시 COMPANY 정보 함께 입력받는 흐름 추가

---

## Phase 2 — 판매자 승인 (ADMIN)

> SELLER가 상품을 등록하려면 ADMIN의 승인이 선행되어야 한다.

- [ ] COMPANY 엔티티 / 레포지토리 / 서비스
- [ ] SELLER 가입 신청 API
- [ ] ADMIN 승인 / 거절 API
- [ ] 승인 상태에 따른 접근 제어 (`COMPANY.status = APPROVED` 확인)

---

## Phase 3 — 카테고리 / 상품

> 장바구니와 주문이 동작하려면 상품이 먼저 존재해야 한다.

- [ ] CATEGORY CRUD (ADMIN만 관리)
- [ ] PRODUCT 등록 / 수정 / 삭제 (SELLER만)
- [ ] PRODUCT 목록 조회 / 상세 조회 (누구나)
- [ ] 카테고리별 필터링, 검색

---

## Phase 4 — 장바구니

> 주문 전에 장바구니가 있어야 한다.

- [ ] CART 자동 생성 (회원가입 시)
- [ ] 장바구니 상품 추가 / 수량 변경 / 삭제
- [ ] 장바구니 목록 조회

---

## Phase 5 — 주문

> 장바구니 → 주문 흐름으로 연결된다.

- [ ] 장바구니에서 주문 생성
- [ ] 주문 목록 / 상세 조회
- [ ] 주문 취소
- [ ] 주문 상태 관리 (SELLER / ADMIN)

---

## Phase 6 — 결제 (포트원 연동)

> 주문이 존재해야 결제가 가능하다.

- [ ] 포트원 SDK 연동 (프론트)
- [ ] 결제 사전 등록 API (백엔드)
- [ ] 결제 검증 API (백엔드 — 포트원 서버에 검증 요청)
- [ ] 결제 취소 API

---

## Phase 7 — 리뷰

> 결제 완료된 주문 건에 한해서만 리뷰 작성이 가능하다.

- [ ] 리뷰 작성 (구매 확인 후 — `ORDER_ITEM` FK로 검증)
- [ ] 리뷰 수정 / 삭제
- [ ] 상품별 리뷰 목록 조회
- [ ] 평점 평균 계산

---

## Phase 8 — 프론트엔드 연동

> 백엔드 API가 어느 정도 완성된 후 프론트를 붙인다.

- [ ] 공통 컴포넌트 (헤더, 레이아웃, 버튼 등)
- [ ] 회원가입 / 로그인 페이지
- [ ] 상품 목록 / 상세 페이지
- [ ] 장바구니 페이지
- [ ] 주문 / 결제 페이지 (포트원 결제창 연동)
- [ ] 마이페이지 (주문 내역, 리뷰)
- [ ] 판매자 페이지 (상품 관리, 주문 관리)
- [ ] 관리자 페이지 (판매자 승인)

---

## 전체 흐름 요약

```
Phase 0  프로젝트 세팅
    ↓
Phase 1  인증 / 회원       ← 기존 코드 보완
    ↓
Phase 2  판매자 승인       ← SELLER가 있어야 상품 등록 가능
    ↓
Phase 3  카테고리 / 상품   ← 상품이 있어야 장바구니 가능
    ↓
Phase 4  장바구니          ← 장바구니가 있어야 주문 가능
    ↓
Phase 5  주문              ← 주문이 있어야 결제 가능
    ↓
Phase 6  결제              ← 결제가 있어야 리뷰 가능
    ↓
Phase 7  리뷰
    ↓
Phase 8  프론트엔드 연동
```
